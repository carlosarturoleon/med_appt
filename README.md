# Medical Appointment Booking

A front-end web application for **StayHealthy**, a non-profit organization dedicated to improving healthcare access in remote areas. This platform connects patients with doctors and specialists for online consultations and medical prescriptions.

## Project Scenario

StayHealthy conducted an extensive survey and found that patients in remote areas struggle to find doctors within a 50-mile radius, or face extremely long wait times. With broadband availability increasing in these areas, this platform bridges the gap between patients and healthcare providers.

## Features (Phase 1)

### Medical Appointments Online
- Schedule appointments with doctors anywhere, anytime
- View available time slots and select preferred times
- Instant consultations without prior appointments
- Appointment reminders and notifications
- Ability to cancel or modify appointments

### Doctor Listing
- Browse available doctors with ratings
- View doctor profiles, ratings, and review comments
- Search for a specific doctor by name

### Consultation Feedback
- Rate consultations on effectiveness, diagnosis, and communication
- Contribute to system transparency and help other patients make informed choices

### Profile Management
- User sign-up to book appointments
- Input and manage personal details (name, address, phone number, age, blood group, medical records)
- Update personal information anytime

### Navigation
- Intuitive navigation with minimal clicks

## Tech Stack
